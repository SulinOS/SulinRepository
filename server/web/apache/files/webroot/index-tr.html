Sulin Web Sunucusu Deneme Sayfası

Bu sayfanın hazırlandığı diğer diller: [İngilizce] [Almanca] [İspanyolca] [Brezilya Portekizcesi]

Bu sayfayı gördüğünüze göre Sulin üzerine Apache Web Sunucusu'nu başarılı bir şekilde kurdunuz demektir. Bu sayfanın bulunduğu dizine kendi sayfalarınızı ekleyip hemen yayımlamaya başlayabilirsiniz.

Sulin, Apache Web Sunucusu'nun güvenli bir şekilde çalışabileceği özgür bir GNU/Linux dağıtımıdır.

Apache Web Sunucusu'nu kullanırken aşağıdaki Apache logosunu kullanabilirsiniz. Sulin ve Apache'yi kullandığınız için teşekkür ederiz!
