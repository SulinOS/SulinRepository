Sie haben erfolgreich den Apache Webserver in Sulin installiert. Ab jetzt ist es möglich lokale HTML Webseiten in dieses Verzeichnis hinzuzufügen.

Sulin, ist eine Linux Distribution, zur freien Nutzung, Änderung und Verbreitung, auf der der Apache Webserver in einer sicheren Umgebung ausgeführt werden kann. Das folgende Apache Logo können Sie für Ihre Webseiten auf dem Apache Webserver benutzen.
